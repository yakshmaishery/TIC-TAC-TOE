# TIC-TAC-TOE
> The TIC-TAC-TOE is simple game. The Game has been build in the **Electron JS** Framework which is used to build the Desktop Based Applications from web development technologies such as **HTML, CSS and JavaScript**

# New Game
> The **New Game** Button will not reset the scores of the X Player and O Player.

# Reset Game
> The **Reset Game** Button will reset the entire game including the scores of the X Player and O Player

## Prerequisites
- Install **Node JS**
- Install **Electron JS** Library

## How to Start
- npm start
